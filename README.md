# MacroDRF
这是一个Django REST framework 开发的自建基础工具,可以按照这些部分代码修修改改可以快速完成你想要的 DRF api后台。

## 实现的功能
- [x] xadmin
- [x] jwt用户认证
- [x] user app
- [x] drf api doc
